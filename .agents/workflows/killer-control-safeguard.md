RULE NAME: KILLER-CONTROL-MASTER-SAFEGUARD

Before making any change, first state:
1) what exact files will be edited,
2) what exact behavior will change,
3) what will remain untouched.

Then make only the minimum required change.

Do not modify design, layout, copy, translations, routes, auth flow, Firebase config, data structure, or unrelated files unless explicitly requested.

If any part of the task is ambiguous or could affect behavior outside the requested scope, stop and ask before editing.
